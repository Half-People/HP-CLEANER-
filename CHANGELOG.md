## [1.0.0] - 2026-06-06

### Added
- Initial public release for Windows x64
- Smart cleanup: system, browser, game, and developer tool categories with scan-before-delete
- Performance tuning: startup items, services, power plans, one-click presets
- Disk health: SMART / NVMe attributes, health score, optional speed and surface tests
- File map: treemap and directory tree for large folders
- Built-in UI languages: zh-TW, zh-CN, en-US, ja-JP
- Cleanup history and in-app diagnostic log
- System tray quick access

### 中文（1.0.0）

- 智慧清理：系統、瀏覽器、遊戲、開發工具等分類，刪除前可掃描預覽
- 效能優化：啟動項、服務、電源配置、一鍵預設
- 硬碟健康：SMART / NVMe、健康評分、可選測速與壞軌測試
- 檔案地圖：Treemap 與目錄樹
- 介面語言：繁中、簡中、英文、日文
