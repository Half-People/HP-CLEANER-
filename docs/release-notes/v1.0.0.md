## HP CLEANER++ v1.0.0 — Initial Release

Windows system management for **gamers** and **developers** — reclaim storage, tune performance, and monitor drive health.

![HalfPeople Studio · HP CLEANER++](https://raw.githubusercontent.com/Half-People/HP-CLEANER-/main/assets/head.png)

### Download / 下載

| Asset | Description |
|-------|-------------|
| **HP_CLEANER++-1.0.0-win64.zip** | Portable package · 可攜版（64-bit, Windows 10+） |
| **HP_CLEANER++-1.0.0-win64.zip.sha256** | SHA256 checksum · 校驗檔 |

Extract and run `HP_CLEANER++.exe`. Keep the `i18n/` folder beside the executable.  
解壓後執行 `HP_CLEANER++.exe`，請保留同目錄的 `i18n/` 資料夾。

完整硬碟健康資料建議**以系統管理員身分執行**。

### Highlights / 本版重點

- **Smart cleanup** — System, browsers, games, dev tools; scan before delete  
  **智慧清理** — 系統、瀏覽器、遊戲、開發工具；刪除前可掃描
- **Performance tuning** — Startup, services, power plans  
  **效能優化** — 啟動項、服務、電源配置
- **Disk health** — SMART / NVMe, health score, optional tests  
  **硬碟健康** — SMART / NVMe、健康評分、可選測試
- **File map** — Treemap and folder tree  
  **檔案地圖** — Treemap 與目錄樹
- **Languages** — zh-TW, zh-CN, en-US, ja-JP  
  **多語言** — 繁中、簡中、英文、日文

---

### System requirements / 系統需求

- Windows 10 or later (64-bit recommended) · Windows 10 或以上（建議 64 位元）
- 4 GB RAM minimum (8 GB+ for large scans) · 至少 4 GB 記憶體
- 1280×720 display · 1280×720 以上解析度
- DirectX 9 compatible graphics · 支援 DirectX 9

---

### Known limitations / 已知限制

- Some tasks require administrator privileges · 部分功能需管理員權限
- Disk surface tests may take a long time · 表面測試可能耗時較長
- Headless / minimal Windows Server may fail to start (D3D9 UI) · 無圖形環境可能無法啟動
