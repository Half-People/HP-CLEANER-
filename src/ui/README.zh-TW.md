# ui

不專屬單一分頁的共用 UI 元件。

## 主要檔案

| 檔案 | 用途 |
|------|------|
| `HUiTheme.h` | Cyber 霓虹色系與間距 token |
| `HUiHistoryList.*` | 可重用捲動紀錄列表 |
| `HUiPathBreadcrumb.*` | 路徑麵包屑 |
| `HUninstallUI.*` | 解除安裝精靈 UI |
